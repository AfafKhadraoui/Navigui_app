# Required Images

Please add the following images to this folder:

## student.png
- An image representing a student (e.g., student studying, graduation cap, books, etc.)
- Recommended size: 80x80 pixels or higher (square format)
- Format: PNG with transparent or white background

## employer.png
- An image representing an employer (e.g., business person, office, briefcase, etc.)
- Recommended size: 80x80 pixels or higher (square format)
- Format: PNG with transparent or white background

## Tips:
- You can use free stock images from sites like:
  - https://unsplash.com
  - https://pexels.com
  - https://www.flaticon.com (for icons)
  - https://iconscout.com

- Or create simple illustrations/icons using tools like:
  - Canva
  - Figma
  - Adobe Illustrator

Once you add the images here, run:
```
flutter pub get
```

Then restart the app to see the changes.
