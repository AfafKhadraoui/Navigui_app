To complete the setup, please download the Aclonica font:

1. Visit: https://fonts.google.com/specimen/Aclonica
2. Click "Download family"
3. Extract the ZIP file
4. Copy Aclonica-Regular.ttf to this fonts folder

OR use this direct link:
https://github.com/google/fonts/raw/main/ofl/aclonica/Aclonica-Regular.ttf

The app is configured to use this font once you add the .ttf file here.
